# Number System Base Converter

A command-line application developed in Python that performs conversions between different number systems using manually implemented conversion algorithms. The project demonstrates the mathematical principles behind Binary, Octal, Decimal, and Hexadecimal conversions.

## Key Highlight

> This project was developed completely from scratch without using Python's built-in conversion functions such as `bin()`, `oct()`, `hex()`, or similar shortcuts. All conversion logic has been implemented manually using mathematical operations and custom algorithms.
>
> ## Development Approach

The primary goal of this project was not only to perform conversions but also to understand and implement the internal working principles of number system conversions from first principles.

## Features

### Supported Conversions

* Decimal → Binary
* Binary → Decimal

## Technical Implementation

The conversion algorithms are implemented manually using:

* Repeated Division Method
* Positional Number System Logic
* Power-Based Calculations
* Character-to-Value Mapping for Hexadecimal Digits
* Custom Validation Algorithms

No built-in number system conversion utilities were used during development.
