# Console Calculator

A simple menu-driven calculator developed using Python. The application allows users to perform various arithmetic operations through a command-line interface while also providing operator information and continuous calculations.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Floor Division
* Modulus
* Exponentiation (Power)
* Operator Information Menu
* Continuous Calculations
* Menu Driven Interface

## Technologies Used

* Python 3
* Match-Case Statements
* Loops
* Conditional Statements

## Project Structure

```text
Console-Calculator/
├── main.py
└── README.md
```

## How to Run

Run the application using:

```bash
python main.py
```

## Available Operators

| Operator | Description    |
| -------- | -------------- |
| +        | Addition       |
| -        | Subtraction    |
| *        | Multiplication |
| /        | Division       |
| //       | Floor Division |
| %        | Modulus        |
| **       | Exponentiation |

## Usage

1. Start the program.
2. Select an option from the menu.
3. Enter two numbers.
4. Choose an operator.
5. View the result.
6. Continue calculations or exit the application.
