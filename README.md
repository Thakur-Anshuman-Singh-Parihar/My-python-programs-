# My Python Programs

A collection of Python projects developed to strengthen programming fundamentals, problem-solving skills, GUI development, and application design.

The repository is divided into two categories:

* GUI Projects
* Terminal Projects

---

# GUI Projects

## Calculator GUI

A Tkinter-based calculator application with:

* Arithmetic Operations
* Keyboard Support
* Calculation History
* Dynamic Font Resizing
* History Storage

**Technologies:** Python, Tkinter

---

## Shutdown App

A system utility application built using Tkinter.

**Features**

* Shutdown System
* Restart System
* Delayed Restart
* Logout Function

**Technologies:** Python, Tkinter, OS Module

---

# Terminal Projects

## Number Base Converter

Manual implementation of number system conversion algorithms.

**Conversions**

* Decimal ↔ Binary
* Decimal ↔ Octal
* Decimal ↔ Hexadecimal

**Highlight**

* Developed from scratch without using `bin()`, `oct()`, or `hex()`.
